Quick Installation

Install any widget with a single command. Each widget comes with complete documentation and examples.

1

Install & Initialize

Download Plumer and initialize your project

wget https://raw.githubusercontent.com/denisdevke/plumer/main/plumer -O plumer
chmod +x plumer
./plumer init
2

Install Widget

Choose and install any widget you need

././plumer install:widget button
3

Start Using

Import and use in your Flutter app

import 'widgets/button.dart';

Custom Installation Path

You can specify a custom folder for widget installation:

././plumer install:widget button lib/components
